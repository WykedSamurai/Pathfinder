# Pathfinder Tabletop Hub

A separate private tabletop roleplay platform project for Pathfinder, D&D, Star Wars Saga, and custom campaigns.

## Version 0.1 Goal

Build the Campaign Hub foundation first:

- Create campaigns
- Select a game system
- Keep each campaign's lore, rules, characters, sessions, and AI context separate
- Prepare modules for live roleplay, rules notes, crafting, and kingdom building

## Supported Systems Planned

- Pathfinder 1E
- Pathfinder 2E
- D&D 5E
- Star Wars Saga Edition
- Custom / homebrew systems

## Planned Modules

- Campaign dashboard
- Character vault
- NPC and location database
- Rules and house-rule notes
- Tabletop etiquette and Session Zero tools
- Live roleplay chat
- Dice roller
- Crafting engine
- Kingdom building tools
- AI assistant with campaign-separated memory

## Local Development

```bash
npm install
npm run dev
```

Then open the local URL shown in the terminal.

## Project Rule

Each campaign is its own separate world. Characters, lore, NPCs, rules, notes, chat, and AI context must not bleed between campaigns.
